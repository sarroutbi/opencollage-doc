\section{Governance \& Community}

\begin{frame}{Open Governance, Open Community}
  \vspace{-0.3cm}
  \scriptsize
  \begin{block}{Core Message}
    \scriptsize
    OpenCollage is built in the open from day one---Apache 2.0 licensed,
    community-governed, with an RFC process that ensures no single vendor
    controls the standard.
  \end{block}

  \vspace{0.3cm}

  \begin{itemize}
    \item \textbf{Apache 2.0 License}: Permissive, enterprise-friendly, no CLA barriers
    \item \textbf{RFC-driven evolution}: All protocol changes go through public proposal and review
    \item \textbf{Conformance suite}: Automated tests verify that adapters meet the standard
    \item \textbf{CNCF trajectory}: Designed for eventual submission as a CNCF sandbox project
  \end{itemize}
