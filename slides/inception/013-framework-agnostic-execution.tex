\section{Framework-Agnostic Execution}

\begin{frame}{Not a Competitor---An Enabler}
  \vspace{-0.3cm}
  \scriptsize
  \begin{block}{Core Message}
    \scriptsize
    OpenCollage does not replace AutoGen, CrewAI, or LangGraph. It provides
    a thin adapter layer so that agents from \textbf{any} framework participate
    as first-class nodes in a shared DAG.
  \end{block}

  \fontsize{6}{7}\selectfont
  \textbf{The Adapter Contract} (what framework authors implement):
  \begin{enumerate}
    \item \texttt{initialize(config)} --- Bootstrap the agent from pipeline YAML
    \item \texttt{execute(input, state)} --- Run the agent and return typed output
    \item \texttt{emit\_telemetry(span)} --- Report execution telemetry via OTLP
    \item \texttt{status()} --- Return current agent state and metadata
  \end{enumerate}
  \vspace{-0.2cm}
  \begin{exampleblock}{}
    \fontsize{6}{7}\selectfont
    Four methods. That is the entire surface area a framework needs to implement
    to join the OpenCollage ecosystem.
  \end{exampleblock}
