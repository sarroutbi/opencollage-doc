\section{Agent Deployment Automation}

\begin{frame}{\fontsize{10}{12}\selectfont GitOps for Agent Pipelines}
  \vspace{-0.2cm}
  \begin{block}{\scriptsize Core Message}
    \scriptsize
    Agent pipelines are code. They should be version-controlled, reviewed,
    tested in CI, and deployed through GitOps---the same way we deploy
    infrastructure and applications today.
  \end{block}

  \vspace{-0.1cm}

  \vspace{-0.1cm}

  \fontsize{6}{7}\selectfont
  \begin{columns}[T]
    \column{0.48\textwidth}
    \begin{block}{\fontsize{6}{7}\selectfont CI Validation Steps}
      \fontsize{5.5}{6.5}\selectfont
      \begin{enumerate}
        \setlength{\itemsep}{0pt}
        \item \textbf{Schema validation}: pipeline YAML conforms to spec
        \item \textbf{Agent resolution}: all referenced agents exist in registry
        \item \textbf{Conformance check}: adapters pass the test suite
        \item \textbf{Dry-run}: execute pipeline with mock inputs
        \item \textbf{Cost estimation}: predict token usage before production
      \end{enumerate}
    \end{block}

    \column{0.48\textwidth}
    \begin{exampleblock}{\fontsize{6}{7}\selectfont Why This Matters}
      \fontsize{5.5}{6.5}\selectfont
      \begin{itemize}
        \setlength{\itemsep}{0pt}
        \item \textbf{Reproducibility}: same pipeline YAML = same behavior
        \item \textbf{Auditability}: every change is a commit with review
        \item \textbf{Rollback}: \texttt{git revert} undoes a bad agent deployment
        \item \textbf{Compliance}: automated policy gates before production
        \item \textbf{Velocity}: deploy agent updates as fast as code
      \end{itemize}
    \end{exampleblock}
  \end{columns}
\end{frame}

\begin{frame}{\fontsize{10}{12}\selectfont Agent Supply Chain Security}
  \vspace{-0.2cm}
  \begin{alertblock}{\scriptsize Core Message}
    \scriptsize
    AI agents execute code, access APIs, and handle sensitive data. Without supply
    chain controls, a compromised or misconfigured agent is an attack vector.
    OpenCollage applies \textbf{software supply chain best practices} to agents.
  \end{alertblock}

      \fontsize{5.5}{6.5}\selectfont
      \begin{itemize}
        \setlength{\itemsep}{0pt}
        \item \textbf{Prompt injection}: malicious system prompts in untrusted agents
        \item \textbf{Tool abuse}: agents with over-permissioned tool access
        \item \textbf{Model drift}: unversioned model updates change behavior silently
        \item \textbf{Data exfiltration}: agents sending context to unauthorized endpoints
        \item \textbf{Dependency confusion}: typosquatting in agent registries
      \end{itemize}
    \end{alertblock}
