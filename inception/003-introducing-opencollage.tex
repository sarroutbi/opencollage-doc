\section{Introducing OpenCollage}

\begin{frame}{OpenCollage: The Universal Orchestration Layer}
  \scriptsize
  \begin{block}{Core Message}
    \scriptsize
    OpenCollage is not another agent framework---it is the open orchestration
    standard that sits \textbf{above} frameworks, enabling heterogeneous agents
    to collaborate through shared definitions, protocols, and telemetry.
  \end{block}

  \vspace{0.2cm}

  \begin{itemize}
    \item \textbf{Open definitions}: YAML/JSON pipeline specs---version-controlled, portable, auditable
    \item \textbf{Framework-agnostic}: Agents from any SDK participate as first-class citizens
    \item \textbf{Observable by default}: Native OpenTelemetry integration for every orchestration step
  \end{itemize}
