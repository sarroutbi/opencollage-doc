      \textbf{\faIcon{layer-group}\ \ Design Decisions:}
      \begin{itemize}
        \setlength{\itemsep}{0pt}
        \item Envelope is \textbf{CloudEvents v1.0}---interoperable with Knative, Azure Event Grid, and any CE-compliant broker
        \item \texttt{trace\_context} propagates W3C Trace Context across the entire DAG
        \item \texttt{schema\_ref} is validated by the schema registry \textbf{before} delivery
        \item \texttt{routing} tells the bus whether to broadcast or unicast
        \item \texttt{data} contains only the typed handoff payload---no framework internals leak
      \end{itemize}
