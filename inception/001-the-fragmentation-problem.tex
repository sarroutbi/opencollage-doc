\section{The Fragmentation Problem}

\begin{frame}{The Agent Landscape Is Fragmented}
  \scriptsize
  \begin{block}{Core Message}
    \scriptsize
    AI agents are powerful individually, but the ecosystem lacks a universal way
    to compose, orchestrate, and monitor heterogeneous agents working together.
  \end{block}

  \begin{itemize}
    \item \textbf{Framework lock-in}: Agents built in one framework cannot collaborate with agents in another
    \item \textbf{No shared protocol}: Each framework defines its own message formats, state schemas, and execution models
    \item \textbf{Reinventing the wheel}: Every team builds bespoke glue code for multi-agent coordination
  \end{itemize}
